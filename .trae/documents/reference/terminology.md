## 特殊处理

### 不翻译的内容
- 代码中的标识符（类名、方法名、变量名）
- API 名称
- NPC 名称（游戏内角色名）
- 地名（使用官方翻译）
- 技能名称（使用官方翻译）

### 需要特别注意的术语
1. **Glamour**: 统一使用"投影"，不用"幻化"
2. **Armoury**: 统一使用"兵装库"，不用"军械库"
3. **Retainer**: 统一使用"雇员"，不用"随从"
4. **Free Company**: 统一使用"部队"，不用"自由公司"
